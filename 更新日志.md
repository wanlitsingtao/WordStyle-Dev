## v2.7 (2026-04-25) - 进度条两级控制优化

### ✨ 新增功能
- ✅ **实现两级进度控制**
  - **第一级**：文件级别，每个文件占 1/n 的进度
  - **第二级**：单文件内部，根据步骤在文件范围内推进
  - 进度单调递增，不会重置或回退
  - 精确反映整体转换进度
  
### 🔧 技术改进
- 修改进度条最大值为 `总文件数 × 100`
- 单文件模式：进度 = (step/7) × 100
- 多文件模式：进度 = (已完成文件数 × 100) + (step/7 × 100)
- 确保进度不超过当前文件的上限
- 使用 `root.after()` 保证线程安全

### 用户体验
- 📊 清楚看到整体进度（例如：3个文件完成了57%）
- 😊 不再困惑“为什么进度条重置了”
- 🎯 准确预估剩余时间

## 如何更新

如果您之前下载了 v1.0 版本并遇到启动错误，请：

1. **重新下载** `doc_converter_gui.py` 文件
2. **或者手动修复**：在文件开头的导入语句中添加 `LabelFrame`

```python
# 修改前
from tkinter import (Tk, Frame, Label, Button, Entry, ...)

# 修改后
from tkinter import (Tk, Frame, Label, LabelFrame, Button, Entry, ...)
```

3. **重新启动程序**即可
