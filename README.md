# Word文档格式转换工具

一个功能完整的Word文档格式转换桌面应用程序，支持样式转换、祈使语气转换和自动插入应答句。

## 功能特性

### 1. 文件选择
- **单文件模式**：选择一个源文档进行转换
- **多文件模式**：批量选择多个源文档进行转换

### 2. 样式识别与映射
- **自动识别**：自动分析源文档和模板文档中使用的所有样式
- **自定义映射**（单文件模式）：用户可自定义源样式到模板样式的映射关系
- **默认规则**（多文件模式）：系统根据内置规则自动进行样式转换

### 3. 样式转换
- 将源文档的样式转换为模板文档的样式
- 保留模板文档的格式样式，不引入源文档样式
- 支持标题、正文、列表等各种段落样式
- 支持表格和图片的样式处理
- 自动调整图片大小以适应页面
- 自动为表格添加边框

### 4. 祈使语气转换
自动将文档中的祈使语气转换为投标人语气：
- `必须` → `将`
- `应` → `将`
- `须` → `将`
- `不得/不应/不可` → `不会`
- `不能` → `无法`
- `投标人` → `本投标人`
- 智能识别例外词汇（如"响应"、"应用"、"不可抗力"等），避免误替换

### 5. 自动插入应答句
选择你需要插入应答句的位置：
- 检测标题后的正文、表格或图片
- 在标题后插入自定义格式的应答句
- 默认内容：`应答：本投标人理解并满足要求。`
- 应答文本和样式可由用户自定义

## 安装说明

### 系统要求
- Windows 7/10/11
- Python 3.7 或更高版本

### 安装步骤

1. **安装Python**（如果尚未安装）
   - 访问 https://www.python.org/downloads/
   - 下载并安装Python 3.7+
   - 安装时勾选"Add Python to PATH"

2. **安装依赖包**
   ```bash
   pip install python-docx Pillow
   ```
   
   或者使用提供的批处理文件自动安装：
   ```bash
   启动转换工具.bat
   ```

## 使用方法

### 方式一：使用批处理文件（推荐）
双击运行 `启动转换工具.bat`，程序会自动检查并安装依赖。

### 方式二：命令行启动
```bash
python doc_converter_gui.py
```

### 操作流程

#### 单文件转换

1. **选择源文档**
   - 点击"浏览..."选择单个源文档
   - 或点击"多选..."选择单个文件

2. **选择模板文档**
   - 点击"浏览..."选择模板文档

3. **配置转换选项**
   - ☑ 进行祈使语气转换（默认勾选）
   - ☐ 使用自定义样式映射（可选，点击后会弹出映射对话框）

4. **配置应答句**（可选）
   - 修改应答句文本（默认为"应答：本投标人理解并满足要求。"）
   - 修改应答句样式（默认为"应答句"）

5. **选择输出文件**（可选）
   - 如不指定，将自动生成文件名（原文件名_converted.docx）

6. **点击"开始转换"**
   - 查看日志区域的转换进度
   - 转换完成后会弹出提示框

#### 多文件批量转换

1. **选择多个源文档**
   - 点击"多选..."按钮
   - 按住Ctrl或Shift选择多个文件

2. **选择模板文档**
   - 点击"浏览..."选择模板文档

3. **配置转换选项**
   - ☑ 进行祈使语气转换
   - （多文件模式不支持自定义样式映射）

4. **点击"开始转换"**
   - 系统会为每个源文件生成独立的输出文件
   - 输出文件名格式：原文件名_converted.docx

## 技术架构

### 核心模块

1. **doc_converter.py** - 核心转换引擎
   - `DocumentConverter` 类提供所有转换功能
   - 样式转换、语气转换、应答句插入

2. **doc_converter_gui.py** - Tkinter图形界面
   - `DocumentConverterGUI` 类实现主界面
   - `StyleMappingDialog` 类实现样式映射对话框

### 依赖库

- **python-docx**: Word文档处理
- **Pillow**: 图片尺寸调整（可选）

## 常见问题

### Q: 为什么图片没有自动调整大小？
A: 请确保已安装Pillow库：`pip install Pillow`

### Q: 多文件模式为什么不能使用自定义样式映射？
A: 多文件模式下，不同源文档可能使用不同的样式，统一映射可能导致错误。建议对需要特殊处理的文件使用单文件模式。

### Q: 如何修改默认的样式映射规则？
A: 编辑 `doc_converter.py` 文件中的 `STYLE_MAP` 字典。

### Q: 转换后的文档格式不正确怎么办？
A: 
1. 检查模板文档是否包含所需的样式
2. 使用自定义样式映射功能手动指定映射关系
3. 查看日志文件（xxx_err.log）了解详细错误信息

### Q: 某些祈使词没有被转换？
A: 系统内置了例外词列表以避免误替换。如需调整，可编辑 `doc_converter.py` 中的例外词列表。
