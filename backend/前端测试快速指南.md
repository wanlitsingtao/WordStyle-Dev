# 🎯 前端测试页面快速使用指南

## ⚡ 3步开始测试

### 第1步：启动后端服务（如果还没启动）

双击运行：`backend\启动后端服务.bat`

或者在命令行执行：
```bash
cd backend
venv\Scripts\python.exe run_dev.py
```

看到以下输出表示成功：
```
📖 API 文档: http://localhost:8000/docs
🔍 ReDoc:    http://localhost:8000/redoc
```

---

### 第2步：打开测试页面

**方式一（推荐）**：双击 `backend\打开测试页面.bat`

**方式二**：直接在浏览器中打开 `backend\test_frontend.html`

---

### 第3步：开始测试功能

#### ✅ 测试流程

1. **检查连接** → 点击"检查健康状态"按钮
   - 应该看到绿色响应框显示服务信息

2. **注册账号** → 填写邮箱和密码，点击"注册"
   - 默认已预填：test@example.com / test123456

3. **登录账号** → 使用刚才注册的账号登录
   - 成功后右上角显示绿色"已登录"徽章

4. **查看信息** → 点击"获取个人信息"
   - 可以看到用户ID、余额等信息

5. **创建订单** → 选择充值金额，点击"创建订单"
   - 会生成一个订单号（演示模式，不会真正扣款）

6. **开始转换** → 输入文件名和段落数，点击"开始转换"
   - 创建转换任务（需要实际的转换逻辑支持）

## ❓ 常见问题速查

### Q: 页面打不开？
**A**: 确保文件路径正确，直接用浏览器打开 HTML 文件

### Q: 点击按钮没反应？
**A**: 
1. 确认后端服务正在运行
2. 按 F12 查看浏览器控制台错误

### Q: 提示 "Failed to fetch"？
**A**: 后端服务未启动或地址不正确

### Q: 登录后刷新页面就退出了？
**A**: Token 应该自动保存，检查浏览器是否禁用了 localStorage

### Q: 想重新开始测试？
**A**: 删除 `backend/wordstyle.db` 文件，重启后端服务

## 💡 小贴士

✨ **快捷键**
- F12：打开开发者工具
- Ctrl+R：刷新页面
- Ctrl+Shift+R：强制刷新（清除缓存）

✨ **查看网络请求**
1. 按 F12
2. 切换到 Network 标签
3. 点击任意按钮
4. 查看请求详情

✨ **清除登录状态**
在浏览器控制台执行：
```javascript
localStorage.removeItem('authToken')
location.reload()
```
